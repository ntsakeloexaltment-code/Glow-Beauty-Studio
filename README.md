# Glow Beauty Studio 💅✨

Welcome to **Glow Beauty Studio**, a simple beauty salon website project designed to showcase services, bookings, and information about the business.

---

## 🌸 About the Project

This is a beginner-friendly website project created using HTML and CSS. It represents a beauty studio where clients can learn about services, view offerings, and navigate through different pages.

---

## ✨ Features

* Homepage with introduction to the studio
* Navigation bar for easy browsing
* Services section
* Booking page (if included)
* About page
* Clean and simple UI design

---

## 🛠️ Built With

* HTML5
* CSS3

---

## 📁 Project Structure

```
Glow-Beauty-Studio/
│
├── homepage.html
├── about.html
├── service.html
├── bookings.html
├── style.css
└── README.md
```

## 🚀 How to Run the Project

1. Download or clone the repository:

   ```bash
   git clone https://github.com/ntsakeloexaltment-code/Glow-Beauty-Studio.git
   ```

2. Open the folder

3. Open `homepage.html` in your browser
